I/O card

PC extension card with multiple I/O

Project Details

  • Hardware Design : Yes
  • Software Design : Yes
  • Test & Certification : No
  • Production : No

(*) The image has been pixelated per client request.

This I/O card can be mounted in a standard PC and uses a simple USB2.0 interface for control.

Multiple cards can be daisy-chained to increase the I/O capabilities of the system. A cross-platform C API can be used for control.

Features:

  • 8 DI
  • 4 DO
  • watchdog functionality
  • buzzer
  • 1 RS485 connection (2/4-wire)
  • firmware can be updated through C API
  • microcontroller based
