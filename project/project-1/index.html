CAN2.0 dongle

A low-cost tool for diagnostic purposes

Project Details

  • Hardware Design : Yes
  • Software Design : Yes
  • Test & Certification : Yes
  • Production : Startup only

This dongle can be used in combination with a certain brand of material handling equipment. When connected to the CAN-bus it unlocks menus for diagnostic purposes.

Features:

  • ARM Cortex-M0 microcontroller
  • CAN2.0 compliant
  • DC-DC converter
  • firmware update via CAN2.0 interface
